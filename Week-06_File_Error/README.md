# Week 06 — File & Error Handling

> **Level:** Intermediate
> **Goal:** Read and write files safely using Python's I/O system, and handle runtime errors gracefully with exceptions.

---

## What You Will Learn

By the end of this week you will be able to:

- Open, read, and write files with `open()` and the `with` statement
- Use **pathlib** and **os.path** to navigate the filesystem
- Catch and handle exceptions with `try / except / else / finally`
- `raise` built-in and custom exceptions with meaningful messages
- Design **custom exception hierarchies** for your own projects
