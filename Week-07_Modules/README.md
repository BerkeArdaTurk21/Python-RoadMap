# Week 07 — Modules & Packages

> **Level:** Intermediate
> **Goal:** Master Python's module system and the most important standard-library modules.

---

## What You Will Learn

By the end of this week you will be able to:

- Use `import`, `from`, `as`, and control what's exported with `__all__`
- Understand `__name__` and `__main__` to write dual-use scripts
- Create and activate **virtual environments** with `venv`
- Navigate the file system with the **`os` module**
- Build **CLI tools** with `sys` and `argparse`
- Work with dates and times using the **`datetime` module**
- Generate random data with the **`random` module**
- Search and manipulate strings using **regular expressions** (`re`)
